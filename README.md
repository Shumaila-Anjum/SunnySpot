# ☀️ SunnySpot - Cabin Booking Management System

SunnySpot is a web-based Cabin Booking Management System developed using PHP and MySQL. The application allows staff members to manage cabin bookings efficiently through a secure login system and an easy-to-use administration panel.

---

## 📌 Project Overview

This project was developed as part of my Web Development studies to demonstrate full-stack web development skills using PHP, MySQL, HTML, CSS, and JavaScript.

The system provides functionality for staff to:

- Securely log in
- View cabin booking information
- Add new cabin bookings
- Update existing bookings
- Delete bookings
- Manage cabin records through an admin dashboard

---

## 🚀 Features

- 🔐 Staff Authentication (Login & Logout)
- 🏡 Cabin Management
- ➕ Add New Cabins
- ✏️ Edit Cabin Details
- ❌ Delete Cabin Records
- 📋 View Available Cabins
- 🎨 Responsive User Interface
- 🗄️ MySQL Database Integration

## 💾 Database

The application uses a MySQL database.

Update the database credentials in:

```

db_config.php

````

Example:

```php
$host = "localhost";
$username = "root";
$password = "";
$database = "sunnyspot";
````

Import the SQL database into MySQL before running the application.

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/SunnySpot.git
```

### Move project to XAMPP

Copy the project into:

```
xampp/htdocs/
```

### Start Services

Open XAMPP and start:

* Apache
* MySQL

### Create Database

* Open phpMyAdmin
* Create a database named:

```
sunnyspot
```

* Import the SQL file.

### Run Application

Open your browser:

```
http://localhost/SunnySpot/
```
